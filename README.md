# ChestLock Addon

Minecraft統合版向けのチェストロック用ビヘイビアパックです。

## インストール

1. `ChestLock.mcaddon` を開く
2. マイクラに自動インポート
3. ワールド設定で「ChestLock」を有効化

## 使い方

ロック：

```
/function lock
```

→ チェストをクリック

解除：

```
/function unlock
```

→ チェストをクリック

※ チートONのワールドで使用してください。

## 対応版

Minecraft Bedrock Edition **1.20〜現在**
